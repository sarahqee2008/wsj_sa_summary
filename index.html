如何每天自动更新此页面？

此页面为静态 HTML，内容需要通过 Claude Code 来刷新。以下是三种方式：

方式一：手动刷新（最简单）

每天打开 Claude Code，输入以下指令：

请帮我更新 WSJ/index.html，抓取今天最新的 WSJ 和 Seeking Alpha 新闻，
更新市场行情数据，保持现有的网格布局和中文摘要格式。

方式二：macOS 定时任务（自动化）

创建一个 shell 脚本配合 cron 或 launchd，每天定时调用 Claude Code CLI：

# 1. 创建更新脚本
cat > ~/update-news.sh <<'EOF'
#!/bin/bash
cd ~/Desktop/Personal\ Website/WSJ
claude -p "更新 index.html: 抓取今天最新WSJ和Seeking Alpha新闻，更新行情数据，保持grid布局和中文摘要"
EOF
chmod +x ~/update-news.sh

# 2. 添加 cron 定时任务（每天早上8点执行）
crontab -e
# 添加这一行：
0 8 * * * ~/update-news.sh

方式三：GitHub Pages + GitHub Actions（推荐，可分享）

将此项目推送到 GitHub，配置 Actions 每日定时运行 Claude Code 更新内容，自动部署到 GitHub Pages，随时随地访问。
